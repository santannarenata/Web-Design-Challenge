Summary: How weather varies with latitude?

The purpose of this project is to analyze how weather varies as you get closer to the Equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset over 500 cities.

After assembling the dataset, we use Matplotlib to plot various aspects of weather vs. latitude. Factors that we looked at include: temperature, cloudiness, wind speed, and humidity. This site provide the source data and visualization created as part of the analysis, as well as explanations and descriptions of any trends and correlations founded.
